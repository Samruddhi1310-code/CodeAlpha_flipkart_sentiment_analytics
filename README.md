#Project Overview

This project focuses on sentiment analysis of Flipkart product reviews to understand customer opinions.

The goal is to classify reviews into Positive, Negative, and Neutral categories.

A rule-based sentiment analysis approach is used without relying on external NLP libraries.

The results help in identifying overall customer satisfaction trends.

Visual representation is used to make sentiment distribution easy to understand.
